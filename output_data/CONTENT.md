# Output Data

Generated by the pipeline. Do not edit manually — all files here are produced by `invoke run`.

| File / Folder | Produced by | Description |
|---|---|---|
| `eeg_features.tsv` | `run-load-eeg` | Harmonised EEG feature table (all participants x all EEG features). |
| `fmri_features.tsv` | `run-load-fmri` | Harmonised fMRI connectivity table (all participants x upper-triangle connectivity values). |
| `results/` | `run-predict` | Prediction results: one JSON/CSV per modality condition (eeg-only, fmri-only, multimodal). |
| `figures/` | `run-notebooks` | Figures generated by the visualization notebooks. |

Note: output files are ignored by Git (see `.gitignore`).
