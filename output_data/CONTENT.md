# Output Data

Generated by the pipeline. Do not edit manually — all files here are produced by `invoke run`.

| File / Folder | Produced by | Description |
|---|---|---|
| `subjects.txt` | `run-intersect` | One subject ID per line — intersection across EEG, fMRI, and phenotype after dropping subjects with missing confound values. |
| `eeg_features.tsv` | `run-load-eeg` | Harmonised EEG feature table (subjects × features). All-NaN columns dropped; sparse NaN imputed with column median. |
| `fmri_features.tsv` | `run-load-fmri` | Harmonised fMRI connectivity table (subjects × upper-triangle connectivity values). Same NaN handling as EEG. |
| `results/{target}/metrics.tsv` | `run-predict` | One row per condition (eeg_only, fmri_only, multimodal): mean/std for all metrics, `p_vs_chance`, paired t-test p-values. |
| `results/{target}/fold_scores.tsv` | `run-predict` | Raw per-fold scores in long format (condition, fold, metric, value). |
| `results/{target}/feature_importances.tsv` | `run-predict` | Per-feature importance (mean/std across outer folds), tagged by modality. Columns: `condition`, `feature`, `modality`, `importance_mean`, `importance_std`. Not produced for SVM RBF (no analytic importance). |
| `scores_by_condition_{target}.png` | `run-notebooks` | Bar chart (mean ± std) with per-fold overlay and p-vs-chance annotations, one per prediction target. |
| `fold_distribution_{target}.png` | `run-notebooks` | Violin plot of per-fold score distribution per condition, one per prediction target. |
| `feature_importance_{target}.png` | `run-notebooks` | Top-20 features per condition (horizontal bar chart), coloured by modality. |
| `modality_importance_{target}.png` | `run-notebooks` | Mean importance aggregated by modality (EEG vs fMRI) for the multimodal condition. |

Note: output files are ignored by Git (see `.gitignore`).
